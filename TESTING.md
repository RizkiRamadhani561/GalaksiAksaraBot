## 🐛 Debugging Checklist

When something breaks:

- [ ] Check bot.log for errors
- [ ] Verify .env file (token, channel, URL)
- [ ] Check Ollama running: `curl http://localhost:11434/api/tags`
- [ ] Check database: `sqlite3 galaxi_aksara.db ".tables"`
- [ ] Test API directly:
  ```bash
  curl http://localhost:11434/api/generate \
    -H "Content-Type: application/json" \
    -d '{"model":"phi","prompt":"test","stream":false}'
  ```
- [ ] Check Telegram token valid
- [ ] Try fallback: disable Ollama, should still work
- [ ] Check Python version: `python3 --version`
- [ ] Check dependencies: `pip list | grep -E "telegram|requests|schedule"`

## 📊 Test Coverage Report

### Unit Tests Needed

Create `test_bot.py`:

```python
import unittest
from db import Database
from personality import PersonalityEngine
from ai_engine import AIEngine

class TestDatabase(unittest.TestCase):
    def test_user_creation(self):
        db = Database(':memory:')
        db.init_user(123, "Test")
        self.assertTrue(db.user_exists(123))
    
    def test_message_save(self):
        db = Database(':memory:')
        db.init_user(123, "Test")
        db.save_message(123, 'user', 'test message')
        history = db.get_chat_history(123)
        self.assertEqual(len(history), 1)

class TestPersonality(unittest.TestCase):
    def test_emotion_detection(self):
        db = Database(':memory:')
        pe = PersonalityEngine(db)
        emotions = pe._detect_emotions("aku sangat sedih")
        self.assertIn('sedih', emotions)

class TestAIEngine(unittest.TestCase):
    def test_fallback_response(self):
        ai = AIEngine()
        response = ai._get_fallback_response("test", "romantis")
        self.assertTrue(len(response) > 0)

if __name__ == '__main__':
    unittest.main()
```

Run:
```bash
python -m unittest test_bot.py -v
```
